# Тренажёр иврита

Игровой тренажёр слов на основе домашних заданий и разборов с учительницей иврита.

- слова, фразы и предложения из личных учебных конспектов в 10 укрупнённых темах
- отдельные семейства глаголов: инфинитив, `он` в прошедшем и `я` (м. р.) в настоящем
- сессии по 20 карточек с интервальными повторениями
- дневная цель, серия занятий и заморозка пропущенных дней
- сад достижений и собственные мнемоники для трудных слов
- два профиля с отдельным прогрессом, синхронизированным между устройствами

## Запуск

Откройте [опубликованный тренажёр](https://kitsunda.github.io/hebrew-trainer/) или запустите локальный HTTP-сервер в папке проекта, например:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`. HTTP-сервер нужен, потому что словарь загружается из отдельного файла `vocab.js`.

## Структура

- `index.html` — интерфейс, стили и логика тренажёра
- `vocab.js` и `lenya-vocab.js` — исходные словари профилей
- `verb-families.js` — очищенные глагольные семейства и три учебные формы
- `manifest.webmanifest` и `icons/` — установка страницы на домашний экран телефона

Формы глаголов сверены с открытым академическим набором Eran Tomer,
*Hebrew Verb Lists* (CC BY 4.0), опубликованным проектом
[NNLP-IL/Hebrew-Resources](https://github.com/NNLP-IL/Hebrew-Resources/tree/master/linguistic_resources/word_lists/hebrew_verbs_eran_tomer).
