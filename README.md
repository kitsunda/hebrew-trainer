# Тренажёр иврита

Игровой тренажёр слов на основе домашних заданий и разборов с учительницей иврита.

- 2917 слов, фраз и предложений, разбитых по 15 темам
- три режима: карточки, выбор варианта, ввод
- дневная цель со стриком и заморозкой пропущенного дня
- коллекция персонажей и ачивки
- прогресс сохраняется в браузере (localStorage), без сервера и аккаунта

Открыть: [index.html](index.html) — статическая страница, работает через GitHub Pages.
