# Тренажёр иврита

Игровой тренажёр слов на основе домашних заданий и разборов с учительницей иврита.

- 2917 слов, фраз и предложений в 10 укрупнённых темах
- сессии по 20 карточек с интервальными повторениями
- дневная цель, серия занятий и заморозка пропущенных дней
- сад достижений и собственные мнемоники для трудных слов
- экспорт и импорт прогресса между браузерами
- без сервера и аккаунта: прогресс сохраняется локально в браузере

## Запуск

Откройте [опубликованный тренажёр](https://kitsunda.github.io/hebrew-trainer/) или запустите локальный HTTP-сервер в папке проекта, например:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`. HTTP-сервер нужен, потому что словарь загружается из отдельного файла `vocab.js`.

## Структура

- `index.html` — интерфейс, стили и логика тренажёра
- `vocab.js` — словарь
